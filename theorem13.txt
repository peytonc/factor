Path A/B Modular Congruence:
Definitions:
    Let P be a sequence of generators applied to the root v_root to reach a parent node v_parent at depth d.
    Let Q_fork ∈ { A, B }.
    Let Q_swap ∈ { A, B } and Q_swap ≠ Q_fork.
    Let R = G_(d+2) * G_(d+3) * ... , where G_j ∈ { A, B, C }, be an arbitrary sequence of subsequent generators applied after the fork.
    Let P_1 = R * Q_fork * P be the original matrix sequence. Matrix multiplication order is reverse of the path order.
    Let P_2 = R * Q_swap * P be the shadow matrix sequence.
    Let v_i ​= P_1 ​* v_root be the state vector for P_1.
    Let w_i ​= P_2 ​* v_root be the state vector for P_2.
    v_i ≡ w_i (mod 2*b_parent) for any depth i>d.
Description:
    When traversing the PPT tree, if a single substitution is made between generator A and generator B at node v_parent, and all subsequent operations remain identical for both branches, then every resulting descendant node on the shadow path P_2 is congruent to the corresponding node on the original path P_1 modulo 2*b_parent. This implies that while the values of the descendants diverge rapidly in magnitude, their residues relative to twice the divergence point's even leg remain locked together for the remainder of the path.

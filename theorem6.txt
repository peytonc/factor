Modular Orbit Graph (Finite State Machine):
Definitions:
    Let M ∈ ℤ be an odd integer modulus such that 3≤M<sqrt(a_target)​​.
    Let v_(d+1) ​≡ G_d * v_d (mod M) define the state transition over the finite ring (ℤ/Mℤ)^3.
    Let Γ = ⟨ A, B, C ⟩ ⊆ GL(3, ℤ/Mℤ) be the group generated by the transition matrices acting on a state v ∈ (ℤ/Mℤ)^3. It represents the entire collection of all possible matrices that can be formed by multiplying A,B,C in any order.
    Let Orbit(M) = { γ·v_root (mod M) : γ ∈ Γ } be the finite set of reachable states starting from the projected root.
Description:
    Projects the infinite descent of the PPT tree into a finite directed graph modulo M. Because the matrices A​,B​,C​ are invertible and act on a finite state space, the modular graph is strongly connected.

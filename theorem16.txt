Geometric Parent Determinism:
Definitions:
    Let p and q be the known integer factors of a PPT node (such as a surrogate node v_S), satisfying 1≤p<q.
    Let λ = q / p be the factor ratio of the child node.
Description:
    Unlike the target semiprime a_target, where the path is obscured by unknown factors, any node with known p and q values allows for immediate backtracking. The specific generator matrix used to create the current node from its parent can be identified deterministically by checking the interval of λ.
    If λ > 3, the current node is an A-child.
    If 2 < λ < 3, the current node is a B-child.
    If 1 < λ < 2, the current node is a C-child.
    If λ = 3, the node is the root v_root and has no parent.

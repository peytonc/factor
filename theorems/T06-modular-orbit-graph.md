# T06 — Modular Orbit Graph (Finite State Machine)

> The modular state graph is strongly connected because the generators are invertible permutations of a finite space.

## Definitions

- Let `M` be an odd integer modulus such that `3 <= M < sqrt(a_target)`.
- Let `v_(d+1) ≡ G_d * v_d (mod M)` define the state transition over the finite ring (Z/MZ)^3.
- Let `Gamma = <A, B, C> subset GL(3, Z/MZ)` be the group generated by the transition matrices acting on a state v in (Z/MZ)^3. It represents the entire collection of all matrices formed by multiplying A, B, C in any order.
- Let `Orbit(M) = { gamma . v_root (mod M) : gamma in Gamma }` be the finite set of reachable states starting from the projected root.

## Statement

Projects the infinite descent of the PPT tree into a finite directed graph modulo M. Because the matrices A, B, C are invertible and act on a finite state space, the modular graph is strongly connected.
